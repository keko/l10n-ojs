Con este pequeno desenvolvemento pretendese facilitar a creación do entorno de traballo para traducir o "Open Journal Systems" (OJS), un software de código aberto para a administración de revistas creado polo Public Knowledge Project.

Esta utilidade, permitirá extraer os ficheiros xml do locale en_US, os ficheiros xml do locale que se pretende traducir, xerar unha memoria de tradución parcial da tradución existente e mediante o uso doutras ferramentas a conversión dos ficheiros xml a outros formatos máis extendidos e que están soportados pola maioría dos aplicativos TAC.

É un desenvolvemento libre, programado en Java e bash scripting, que se apoia noutras ferramentas, principalmente en po4a. Está en desenvolvemento, non é perfecto xa que non vai cubrir todos os casos pero servirá de axuda para traducir OJS con certas limitacións.
